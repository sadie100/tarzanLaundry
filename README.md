## 기획 의도

- 타잔, 세탁을 부탁해는 크래프톤 정글 시설 내 공용 세탁기의 효율적인 운영을 위해 기획한 **세탁기 예약 시스템**입니다.
- 오늘 혹은 내일의 원하는 시간대에 원하는 호수의 세탁기와 건조기를 예약해 보세요.

## 주요 기능 소개

- 로그인, 회원가입
- 현재 존재하는 예약 조회 기능
- 원하는 기기 예약 및 예약 취소 기능
- 내 예약 조회 기능
